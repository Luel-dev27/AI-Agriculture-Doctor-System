# Frontend

React + Vite scaffold for the AI Agriculture Doctor UI.

## Structure

- `public/` static assets (`favicon.ico`, template `index.html`)
- `src/app/` application shell and hash-based router
- `src/pages/` feature pages
- `src/components/` reusable UI components
- `src/services/` API service wrappers
- `src/hooks/`, `src/context/`, `src/utils/`, `src/assets/` placeholders for future implementation

## Run

```bash
npm install
npm run dev
```
